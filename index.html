Gary Campbell

I am a professional software engineer and game developer based in Belfast, Northern Ireland.

Specialising in using the Unity Game Engine to create bespoke and unique XR experiences, I have also been able to develop my full-stack capabilties with WebGL, Javascript/Typescript, ReactJS, ThreeJS and Node.js.

  • Experience with large & small scale projects and meeting deadlines.

Project Work

Selection of completed personal and professional projects.

Project Title: Escape The Night

A classic runner style game, with the goal being to avoid the oncoming obstacles, repel attacking drones, and reach the end before the timer runs out.
Focus on ambiance and lighting to add to oppressive atmosphere.
Achieved with creating custom models and optimising scene with pre-baked lightmaps on modular prefabs. Built in URP.

Previous Client Work

Project Title: Demonised

A non-linear VR Experience created for an up-and-coming 'Witches of Island Magee' tourist attraction in partnership with Ulster University
The project involved the creation of numerous environments in which the user may explore and interact with to learn more about the history of Island Magee.
Built using the OpenXR framework for cross-platform support and URP.

Project Title: SiteIT

Linear VR training simulator focused on Health & Safety procedures within an active building site. Experience contained a variable number of hazards in which the user had to identify and catagorise in order to pass their initial safety training course.

Project Title: Calf-House Builder

WebGl based application allowing for users to create a custom calf-house, this application also allowed for users to get a general view for capacity, layout, costings for materials and the effects that different material properties would have on the building, with the option to export this as a PDF.

Live Project Link

Project Title: Fall-Awareness Trainer

Proof of concept, in partnership with University of Manchester, this application is proposed to be used primarily for healthcare staff within elderly care homes, for those staff to identify and remedy trip hazards and other issues to minimise the risk of elderly fall incidents.
